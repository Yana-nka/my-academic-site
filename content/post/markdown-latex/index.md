## Что такое Markdown?

Markdown — это лёгкий язык разметки, созданный Джоном Грубером в 2004 году. Он позволяет оформлять текст простыми символами, которые легко читаются в исходном виде.

## Основные элементы

**Заголовки:** `# H1`, `## H2`, `### H3`

**Выделение:** `*курсив*`, `**жирный**`, `` `код` ``

**Списки:**
- Маркированные: `* пункт` или `- пункт`
- Нумерованные: `1. пункт`

**Ссылки:** `[текст](url)`

**Изображения:** `![alt](url)`

**Таблицы:**
```
| Столбец 1 | Столбец 2 |
|-----------|-----------|
| Ячейка    | Ячейка    |
```

**Код:** тройные обратные кавычки с указанием языка

## LaTeX

Для математических формул используется LaTeX-синтаксис:

- `$E = mc^2$` — встроенная формула
- `$$\\sum_{i=1}^{n} i = \\frac{n(n+1)}{2}$$` — выносная формула

## Markdown в научной работе

1. Оформление README на GitHub
2. Написание документации
3. Ведение блогов (как этот сайт на Hugo)
4. Создание презентаций (через Pandoc)

## Заключение

Markdown — незаменимый инструмент для любого IT-специалиста. Он прост, удобен и поддерживается везде.
